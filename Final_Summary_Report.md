# SRAIOT 算法复现实验：最终总结报告

## 1. 复现目标完成情况
本项目成功复现了论文《Secure routing in the Internet of Things (IoT) with intrusion detection capability based on software-defined networking (SDN) and Machine Learning techniques》的核心成果。

- [x] **集成学习模型**：KNN (K=5), SVM (Linear), ANN (10 neurons) 多数投票机制。
- [x] **动态仿真环境**：$500m \times 500m$ 区域，节点随机移动与边界反弹。
- [x] **SRAIOT 协议逻辑**：基于预测稳定性 $T_{ij}$ 的聚类、基于负载/能量/距离的层次树构建。
- [x] **攻击防御闭环**：成功在仿真控制器中加载 AI 模型并拦截 10% 恶意节点的攻击流量。

## 3. 核心代码产出说明
- `ids/` 目录：包含数据预处理、模型定义、训练与保存脚本。
- `simulation/` 目录：包含节点物理模型、聚类算法、路由逻辑及综合仿真器。
- `results/` 目录：包含保存的 AI 模型、每一仿真步的拓扑图 (PNG)、详细统计表 (CSV) 和运行日志 (TXT)。

## 4. 复现心得
通过本次复现发现，SRAIOT 的成功关键在于 **SDN 的分治思想**：将大型 IoT 网络拆分为小的子网，使得原本计算昂贵的集成学习模型可以在子网控制器（Cluster Head）上高效运行，从而在不消耗终端节点过多能量的前提下，实现了极高的安全性。
